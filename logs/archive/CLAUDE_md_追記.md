## 農業エージェント（2026-06-12 新設）

農業EC・農業補助金サポートの2軸で農業事業を担う。詳細は `agents/農業/` を参照。

```
agents/農業/
├── 概要.md          # 役割・EC設計・補助金設計・絶対ルール
├── context.md       # 農業事業視点の最新スナップショット
├── 2026-06-12_補助金サポート本気調査.md   # 補助金10種・相場・法律・競合・シミュ
├── 2026-06-12_補助金設計壁打ち.md         # 25%設計の4AI壁打ち
└── 2026-06-12_報酬設計壁打ち.md           # EC30%・補助金25%・優先順位の4AI壁打ち
```

### 農業エージェントの絶対ルール

1. **行政書士なしで農家への補助金サポートを開始しない**（2026年1月改正行政書士法。違反=1年以下の拘禁刑）
2. **EC・補助金の報酬設計をスケ・さとな・なのかに提示する前に1on1で合意を取る**
3. **農業事業はコーチング月300万安定の後にフルスケール。借金完済前は「仕込み期間」**
4. **さとな・なのかへの収益シミュレーションは3段階（月商50万・200万・600万）で渡す。最大値だけ見せない**
