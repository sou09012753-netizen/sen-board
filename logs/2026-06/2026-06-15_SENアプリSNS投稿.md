## ストーリー（5枚構成）

スマホ縦画面。1枚5〜8秒で読めるテキスト量。背景は黒か白で統一。

---

### STORY 1 / 5　【問題提起】

```
コーチングで
「現状を話してください」

って聞いてみて

どれだけ時間が溶けてる？
```

> デザイン指示: 黒背景・白テキスト・中央配置。フォントは太め。

---

### STORY 2 / 5　【核心】

```
問題は聞き方じゃない。

相手が自分のことを
言語化できていないだけ。
```

> デザイン指示: 同上。「言語化できていないだけ」に黄緑のアンダーライン。

---

### STORY 3 / 5　【解決策の提示】

```
セッションの前に
これを渡すようにした。

AI が 22 の問いで
クライアントの本音を引き出す
自己分析アプリ。
```

> デザイン指示: アプリのスクリーンショットを背景に薄く敷くと映える。

---

### STORY 4 / 5　【アウトプット説明】

```
終わったら出てくるもの

📄 言語化カード
　→ 自分でも知らなかった言葉

💬 持ち続ける問い
　→ 次のセッションの起点

⚡ 今週の一手
　→ 気づきを行動に変える
```

> デザイン指示: 白背景・黒テキスト。リスト形式でクリーンに見せる。

---

### STORY 5 / 5　【CTA】

```
コーチング・採用・チームに
使ってみたい人

プロフィールのリンクから
アクセスできます。

↑ リンクはここ
```

## フィード投稿

### パターンA【問題提起型】

**キャプション:**

```
コーチングで一番時間が溶けるの
「現状を話してください」の最初の10分だと思う。

クライアントは嘘をついてるわけじゃない。
自分のことを言語化できていないだけ。

だから作った。

AIが22の問いと深掘りで
クライアントの本音を引き出す自己分析アプリ。

セッション前に渡すと
初回から全然違う話ができる。

プロフィールのリンクから使えます。
コーチ・採用・チーム管理してる人に届いてほしい。

#コーチング #自己分析 #コーチ #起業家 #経営者
```

---

### パターンB【ビフォーアフター型】

**キャプション:**

```
Before：
「自分が何をしたいかわからなくて」
「なんとなく変わりたいんですが」
→ セッション中ずっとここにいる

After：
「動く条件と止まる条件が言語化された状態でくる」
→ 初回から深い話ができる

この差を作ったのが
セッション前に渡す自己分析アプリ。

AIが深掘りしてくれるから
自分では気づかなかった答えが出てくる。

プロフィールから無料で試せます。

#コーチング #コーチ #自己分析ツール #人材育成 #起業
```

---

### パターンC【ストーリー型・共感から入る】

**キャプション:**

```
正直に言う。

コーチング始めた頃、
クライアントの本音を引き出せてる自信がなかった。

質問の技術とか、傾聴の練習とかより先に
「そもそも相手が言語化できていない」
という問題があることに気づいた。

だから作った。

セッション前に渡す自己分析アプリ。
AIが22問＋最大3回の深掘りをしてくれる。

まとめに出てくるのはこれ。

📄 言語化カード（本人も知らなかった自己像）
💬 持ち続ける問い（次のセッションの起点）
⚡ 今週の一手（気づきを行動に変える）

コーチや採用担当、チームリーダーに使ってほしい。
プロフィールのリンクから無料で試せます。

#コーチング #自己分析 #コーチ #人材育成 #23歳経営者
```

## 使い方のメモ

| 素材 | 使いどころ | ゴール |
|------|----------|--------|
| ストーリー5枚 | 週1〜2回。5枚をセットで流す | プロフから流入させる |
| パターンA | フィード投稿の初回 | 問題認識させる |
| パターンB | 2回目以降の投稿 | 具体的なビフォーアフターで刺す |
| パターンC | ストーリー性で深く読ませたい時 | 共感→信頼→行動の流れ |

**ハッシュタグの使い分け**
- 毎回変えなくていい。#コーチング #自己分析 は固定で入れる
- ターゲットに合わせて #経営者 #採用担当 #チームビルディング を追加
