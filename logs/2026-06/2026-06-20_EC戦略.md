# 🔵 コンテキストAI｜事実を並べる（調査ベース）

## JA・市場出荷の手取り実態

農産物の流通経路別に「農家が最終小売価格の何割を手にするか」を整理する。

### 流通経路別 農家手取り（最終小売価格に対する割合）

| 流通経路 | 農家手取り（小売価格比） | 中間マージンの内訳 |
|---------|---------------------|----------------|
| JA共販・市場出荷 | **約20〜40%** | JA手数料・市場手数料・仲卸・小売がそれぞれ抜く |
| JA直売所（道の駅等） | 約65〜80% | 直売所手数料15〜25% |
| 市場経由（卸売市場） | 約25〜45% | 出荷団体3〜5%+市場5.5%+仲卸+小売 |
| 産直EC（食べチョク等） | 約80〜85% | プラットフォーム手数料15〜20% |

### JA・市場流通のマージン構造（実数の根拠）

野菜の標準的な流通コスト構造（農林水産省「食品流通段階別価格形成調査」ベース）：

| 段階 | 取り分（小売100円あたり） |
|------|----------------------|
| 生産者（農家） | **25〜35円** |
| 出荷・集荷団体（JA等） | 8〜10円 |
| 卸売業者（市場） | 5〜7円 |
| 仲卸業者 | 12〜18円 |
| 小売業者 | 30〜40円 |

**重要な事実：**
- 野菜の場合、生産者手取りは小売価格の**おおむね3割前後**が農水省調査の平均像
- JAの販売手数料そのものは「販売額の2〜3%」程度だが、これは**市場での競り値（卸売価格）に対しての手数料**
- 問題は「JA出荷価格＝卸売価格」自体が小売価格の3〜4割しかない点。ここから手数料が引かれる

### 具体例（数字で示す）

**ケース：トマト1kgが店頭で500円で売られている場合**

| 段階 | 金額 |
|------|------|
| 店頭小売価格 | 500円 |
| 仲卸→小売の卸値 | 約330円 |
| 市場の競り値（農家の出荷価格相当） | 約180〜200円 |
| JA手数料等を引いた農家手取り | **約160〜175円** |
| **農家手取り率** | **32〜35%** |

→ つまり**500円で売られているトマトで、農家の懐に入るのは約165円**。

# 🔴 否定AI｜徹底的に穴を叩く

## 穴①：6.8倍の比較はフェアじゃない（同じ土俵じゃない）

JAの165円/kgは「B級含む全量・無選別・即出荷」の数字。
SEN-ECの1,120円/kgは「ギフト品質・2kg化粧箱・厳選品」の数字。

**比較対象がズレている。** 農家がEC用に出せるのは収穫量の一部（秀品・上位等級）だけ。残りはどのみちJAに流す。

→ **正しい比較は「EC化できる分（全体の2〜3割）の手取り増分」で語るべき。** 「全量が6.8倍」と言うと農家に嘘をつくことになり、信頼を失う（SENの「嘘を入れない」ルール違反）。

## 穴②：販売数量のリアリティがない

JAは「作れば全部引き取る」。EC は「売れた分しか金にならない」。

- リール→EC導線で本当に2kg・3,200円のギフトが月に何箱売れるのか？

- 食べチョク等の先行プラットフォームでも、多くの農家の月商

# 🔵 コンテキストAI｜事実と相場を並べる

## 各論点の業界相場・法的事実

### 委託形態の選択肢（法的に3パターンある）

| 形態 | SENの立場 | 売買契約の当事者 | 食品表示責任 | リスク |
|-----|----------|--------------|-----------|-------|
| **A. 販売委託** | 販売者（SENが売る） | SEN⇄消費者 | **SENが負う** | 高 |
| **B. 仲介・媒介** | 場を貸すだけ | 農家⇄消費者 | 農家が負う | 中 |
| **C. 購入者代理** | 消費者の代理 | 農家⇄消費者 | 農家が負う | 中 |

→ 食べチョク・ポケマルは **B（CtoC仲介型）** を採用。だから手数料が安い（農家が責任を持つ）。
→ SENが30%取るなら **A寄り（運営が前面に立つ）** に見えるが、在庫を持たないなら実態はBに近い。**この食い違いが法的爆弾。**

### 取り分相場（前回壁打ちの再掲＋EC事実）

| プラットフォーム | 農家手取り | 責任の所在 |
|--------------|---------|---------|
| 食べチョク | 84% | 農家（仲介型） |
| ポケマル | 85.5% | 農家（仲介型） |
| ふるさと納税系 | 50〜60% | 運営寄り |
| **SEN** | **70%** | **未定義（ここが問題）** |

→ 30%は「仲介型」としては高すぎる。「運営代行型（集客・撮影・ブランディング・CS全部やる）」としてなら妥当圏。
→ **手数料率の高さ＝SENが負う責任の量。30%取るなら相応の責任を負う設計になる。**

### 食品のクレーム・表示の法的事実

- **食品表示法**：生鮮食品でも「名称・原産地」表示義務。加工品はさらに重い。表示義務者は「販売者」。
- **食品衛生法**：食中毒が出た場合、販売形態により責任主体が変わる。ネット販売でも保健所の管轄。
- **PL法（製造物責任）**：生鮮農産物は「未加工」なら対象外だが、カット・加工した瞬間に対象化。
- **特定商取引法**：通販は返品・表示ルールあり。EC運営者（SEN）が表記義務を負う可能性大。

→ つまり**「在庫を持たない＝責任も持たない」ではない。** 表示・特商法表記はEC運営者に来る。

# 🔴 否定AI｜契約前に潰すべき致命的な穴

## 穴①：「委託販売」と言いながら実態は「売買仲介」では？法的性質が曖昧

EC前提を読むと「注文が来たら農家が直接発送（産直ドロップシッピング）」とある。これは法的には3パターンに分かれ、**どれを選ぶかで責任・税務・リスクが全部変わる**。

| モデル | 売買契約の当事者 | 食品表示・PL責任 | SENの立場 |
|-------|--------------|--------------|----------|
| A. 委託販売（受託者販売） | SEN ⇔ 消費者 | **SENが負う** | 販売主体 |
| B. 仲介・媒介 | 農家 ⇔ 消費者 | 農家が負う | 場を提供するだけ |
| C. 買取転売 | 農家⇔SEN、SEN⇔消費者 | SENが負う | 在庫リスクあり |

「在庫を持たない」「農家が直送」なら実態はBの**仲介（媒介）**に近い。だが「SENのECサイトで決済」「SENが集客・ブランディング」する場合、消費者から見たら売主はSENに見える。**外観上の売主と契約上の売主がズレると、トラブル時にSENが矢面に立つ。**

→ 契約書で「SENは販売を媒介するプラットフォーム事業者であり、売買契約は農家と消費者の間に成立する」のか「SENが販売主体」なのかを**第1条で明確に定義しないと、全条項が宙に浮く。**

## 穴②：食品事故（食中毒・異物混入）時の責任分界が抜けると会社が飛ぶ

農産品・生鮮食品は**食中毒・カビ・腐敗・異物混入のリスクが構造的に高い**。クール便の温度管理ミス、収穫後の鮮度劣化、アレルゲン表示漏れ——これらが起きたとき誰が賠償するか。

- 製造物責任法（PL法）：加工食品は「製造物」に該当しうる。生鮮はグレーだが、カット野菜・加工品は対象。
- SENが「販売者」として表示されていれば、PL法上の**表示製造業者**として責任を負う可能性。

→ 契約書に「品質・安全性・表示の責任は農家が負う」「農家起因の事故はSENが消費者に賠償した後、農家に求償できる」条項が無いと、**SENが全額被る。**しかも個人農家に資力が無ければ求償しても回収不能。

## 穴③：個人農家相手だと「下請法・独占禁止法（優越的地位の濫用）」リスク

SENが集客・ブランディング・ECを握り、農家は商品を出すだけ。この構造で**手数料30%**を取り、さらに「安売り禁止」「他社EC出品制限（専属）」などを課すと、立場の弱い個人農家に対する**優越的地位の濫用**と評価されうる。

- 一方的な手数料変更条項
- 返品・廃棄リスクの農家への押し付け
- 専属義務（他で売るな）

→ これらを盛り込みすぎると公取委マターになりうる。**「30%」の対価性（SENが何を提供するか）を契約書で具体的に列挙しないと、ただの中抜きに見える。**

## 穴④：手数料30%の発生タイミング・返金時の扱いが未定義

- 消費者が返品・キャンセルしたら、農家に払った分の手数料はどうなる？
- 消費者がクレジットカードのチャージバックをしたら？
- 決済はSENが受けるのか、農家が受けるのか（**資金決済法・収納代行**の論点）

→ **SENが消費者から決済を一括代行する場合、「収納代行」として資金決済法のグレーゾーンに入る。**金額が大きくなると登録が必要になる可能性。これは専門家確認必須。

## 穴⑤：発送・配送事故の責任が「農家直送」だと曖昧

農家が直接発送する＝**SENは配送品質をコントロールできない**。なのに消費者から見た窓口はSEN。配送遅延・破損・誤配のクレームは全部SENに来るが、原因は農家の梱包・発送ミス。

→ 「発送・梱包・配送手配の責任は農家」「配送事故の一次対応窓口と最終責任の分界」を明記しないと、SENがクレーム処理の無限地獄に落ちる。

## 穴⑥：契約解除後の「顧客データ」の帰属

SENがSNS・ECで集めた顧客は誰のもの？農家が「もう抜ける」と言ったとき、その顧客リストを持って独立されたらSENのビジネスモデルが崩壊する。

→ **顧客データ・販売データの帰属をSENに帰属させる条項**が無いと、農家に育てた顧客ごと逃げられる。
