## ■ 事前調査：JA・市場出荷の手取り実数

### A. JA出荷（共販）の手取り構造

JAに出荷した場合、農家の手取りから差し引かれるコストは以下の通り。

| 費用項目 | 相場（販売価格に対する比率） | 備考 |
|---------|--------------------------|------|
| 販売手数料（共販手数料） | 8〜12% | JA・品目によって差がある |
| 出荷規格・選別費用 | 2〜5% | 等級・規格外弾きのコスト |
| 輸送・集荷費用 | 2〜4% | 地域によって差がある |
| 箱・資材費 | 2〜5% | 農家負担の梱包材 |
| 農協共済・組合費 | 1〜2% | 組合員費用 |
| **合計控除** | **15〜28%** | |
| **農家手取り** | **72〜85%（販売価格に対して）** | |

ただし「販売価格」はJAが市場に卸した価格であり、消費者が支払う価格ではない。

**消費者価格に対する農家手取り率（JA経路）**：

```
消費者価格 → 小売（スーパー等）の利益 → 卸・仲卸の利益 → 市場手数料 → JA手数料 → 農家
```

| 流通段階 | 取り分 |
|---------|-------|
| 小売（スーパー等） | 25〜35% |
| 卸・仲卸 | 10〜15% |
| 市場手数料 | 5〜7% |
| JA手数料 | 8〜12% |
| 輸送・資材等 | 5〜10% |
| **農家手取り（消費者価格ベース）** | **25〜40%程度** |

> **数字の根拠**：農林水産省「食料の生産・流通・消費に関する統計」（2023年）では、野菜・果物の農家手取りが消費者価格の20〜40%に留まるとされている。品目によって差が大きいが、葉物野菜では20%台、果物では30〜40%台が多い。

## 論点1｜委託形態（何の契約を農家と結ぶか）

### 選択肢の整理

| 形態 | 内容 | SENにとっての法的リスク |
|------|------|----------------------|
| **販売委託契約** | 農家が販売を委託。SENは代理人として販売する | 農家の代理人になるため、農家との責任範囲が曖昧になりやすい |
| **業務委託契約（集客・EC運営代行）** | 農家がEC出店の主体。SENは集客・運営を請け負う | SENは「出店支援業者」の立場。売上の分配は業務委託料として設計 |
| **特定委託取引（産直ドロップシッピング）** | 農家が直接消費者に発送。SENは注文の仲介・集客のみ | 消費者との売買契約が農家とSENのどちらに帰属するかを明確にする必要がある |

### 穴（否定AI視点）

販売委託と位置付けると、特定商取引法における「通信販売の販売業者」がSENになるリスクがある。その場合、返品・返金・クレーム対応の法的義務を全てSENが負う。農家が直接発送するにもかかわらずSENが販売責任を持つ構造は、クレーム時に致命的。

### 推奨案

**「業務委託契約（集客・EC運営代行）+ 農家が販売主体」の構造を採用する。**

- 消費者との売買契約者 = 農家
- SENの立場 = 集客・プラットフォーム運営・注文処理代行の受託業者
- SENへの報酬 = 月商30%の業務委託料（成果報酬型）
- 特定商取引法の表記 = 農家名・農家住所を販売者として記載

この構造であれば、SENは「販売代理人」ではなく「インフラ・集客事業者」として機能し、消費者クレームの一次責任を農家が持つ設計になる。

## 論点2｜取り分割合（30%の正当性と設計）

### 市場との比較（数字根拠あり）

| プラットフォーム | 農家手取り | 手数料 | 集客 |
|---------------|---------|-------|------|
| ポケットマルシェ | 85% | 15% | プラットフォーム任せ |
| 食べチョク | 84% | 16% | プラットフォーム任せ |
| BASE（自力出店） | 約93% | 約7% | 農家が自力 |
| **SEN設計** | **70%** | **30%** | **SENが担う** |

SENの30%は「集客 + EC運営 + ブランディング + SNS制作」を含む。プラットフォーム手数料16%と集客代行費10〜20%の合算と見れば26〜36%になるため、30%はギリギリ合理的な水準。

### 穴（否定AI視点）

**農家に「なぜ30%なのか」を説明できるか。**

現状、SENに農産品EC実績はゼロ。スケのリール1本250万の実績がコーチング商品か農産品かが不明であり、農産品ECの集客力の証明になっていない。

また、月商がゼロの間（立ち上げ期3〜6ヶ月）も農家はSENのためにSNS撮影・商品準備の工数を割く。農家にとってのリスクを、30%という数字だけで正当化するのは弱い。

### 推奨案

**段階的報酬制を採用する。**

| 月商 | SEN手数料 | 農家手取り | 農家への説明 |
|------|---------|---------|------------|
| 〜50万 | **25%** | 75% | 実績構築期 |
| 50〜200万 | **28%** | 72% | 成長期 |
| 200万〜 | **30%** | 70% | 本格稼働期 |

理由:

1. 農家にとって「まず試せる」設計になる
2. 実績ゼロの段階で30%を要求することへの正当性リスクを回避できる
3. 売上が増えるほどSENの貢献度が上がる構造と説明できる

**ただし**: 段階制にすると農家が「50万以上売れたらSENより安い他社に乗り換えよう」と考えるリスクがある。そのため専属性（論点7）とセットで設計すること。

## ① 特定商取引法（特商法）

### 適用条件
通信販売（インターネットEC）に該当。農産物の販売に特商法は**完全適用される**。

### 表示義務（特商法第11条）

ECサイト（または申込み画面）に以下を**必ず表示**しなければならない。

| 表示項目 | 内容 | ドロップシッピング時の論点 |
|---------|------|----------------------|
| 販売業者の氏名または名称 | 法人名（SEN）または代表者名 | **SENか農家か、どちらが「販売業者」か**の定義が必要 |
| 住所 | 事業所の住所（P.O.Boxや私書箱は不可） | 仙石の自宅住所を公開したくない場合の対処が必要 |
| 電話番号 | 問い合わせ可能な番号 | 農家が直接発送するが、クレーム窓口はSEN側か農家か |
| 販売価格 | 消費税込みの総額表示 | 送料が農家ごとに異なる場合の対処 |
| 送料 | 送料が別途かかる場合は明示 | 農家の産地・商品によって送料が異なるケースに対応できるか |
| 支払方法・時期 | クレジットカード・銀行振込等 | SENが決済を受けるのか農家が受けるのか |
| 引渡し時期 | 何日以内に発送するか | 農家が発送主体の場合、SENが管理できるか |
| 返品・キャンセル規定 | 条件・期間・費用負担 | 生鮮食品の返品不可ルールの明示方法 |

### 特に問題になりうる点

**「販売業者」の定義問題**:
- ドロップシッピングでSENが注文を受け、農家が発送する場合、法的な「販売業者」はSENになる可能性が高い
- 農家が「販売業者」として特商法表示をする形にするか、SENが「販売業者」として表示するかで責任の所在が変わる
- どちらの設計にするかを専門家と確認する必要がある

**住所表示問題**:
- 法人住所が確定しているか確認。ウェブサイト住所変更タスクが未完了であれば、現在の表示住所が実際の法人住所と一致しているかを確認する

**返品・キャンセル**:
- 生鮮食品は原則として返品不可の特約を設定できる（特商法第15条の3）
- ただし「商品が届かない」「腐敗していた」等のケースへの対応規定が必要
- 農家が発送主体の場合、農家の対応品質にSENが引きずられるリスクがある

## 事前分析：穴の洗い出し（否定AI）

契約書を書く前に、この事業モデルが抱える法的・実務的リスクを先に明示する。

### 致命的リスク①：特定商取引法の適用

ECで消費者に直接販売する場合、**特定商取引法（特商法）** が適用される。

| 論点 | 内容 |
|------|------|
| 販売業者は誰か | SEN法人か・農家か・どちらが「販売業者」として特商法の義務を負うか |
| 返品・返金の責任 | 産直ドロップシッピングの場合、消費者クレームの一次窓口はSENか農家か |
| クーリングオフ | 食品は適用外だが、生鮮品の「届いたら腐っていた」への対応責任の所在 |

**SENがプラットフォームとして「販売業者」になる設計**なら、特商法の表記義務・返金対応はSENが負う。農家に転嫁できない。これを契約書で明確にしないと後で揉める。

### 致命的リスク②：食品衛生法・農産物販売の規制

| 論点 | 内容 |
|------|------|
| 食品表示法 | 産地・内容量・消費期限等の表示義務は農家にあるが、ECサイト掲載もSENの責任 |
| 農産物直売 | 農家が直接発送する場合、農家は「通信販売業者」として届出が必要な場合がある |
| 精算農産物 | 米の場合は食糧法の規制あり（農産物は米・麦などで別途規制あり） |

### 致命的リスク③：消費税・インボイスの処理

| 論点 | 内容 |
|------|------|
| 代金の流れ | 消費者→SEN→農家 の場合、SENが課税事業者なら全額に消費税処理が必要 |
| インボイス | 農家がインボイス未登録の場合、SENは仕入税額控除ができない |
| SEN手数料の消費税 | SENが農家から30%手数料を受け取る構造では、手数料に消費税がかかる |

この問題は**税理士確認が必須**。契約書の素案には「消費税・インボイス処理は別途税理士の指示に従う」と明記するにとどめる。

### 致命的リスク④：産直ドロップシッピングの「販売委託」か「代理販売」か

法的な性質をどちらに設計するかで、税務・責任・消費税の扱いが全て変わる。

| 設計 | 概要 | SENの立場 |
|------|------|---------|
| **販売委託** | 農家の商品をSENが代わりに売る。所有権は農家のまま | 受託販売業者 |
| **買取→転売** | SENが農家から商品を買い取り消費者に販売 | 仕入れ→販売 |
| **代理販売（取次）** | SENは場所を提供するだけ。農家が直接消費者と契約 | プラットフォーム |

産直ドロップシッピングは「販売委託」が最も実態に近いが、**どの設計にするかを弁護士と決定してから契約書に落とす必要がある**。今回の素案は「販売委託」前提で作成するが、専門家確認が必須。

| トラブル | 発生頻度 | 責任の設計 |
|---------|---------|---------|
| 商品の品質不良 | 高（農産物の特性上） | 農家責任か・SEN責任か |
| 発送遅延 | 中 | 農家責任。ただしSENがお詫び対応するか |
| 梱包ミス | 中 | 農家責任 |

| 食物アレルギー表示漏れ | 低 | 最悪の場合、健康被害。表示義務違反 |
