# SEN Board 壁打ち
## 議題：農業補助金サポートビジネスの報酬設計25%は適切か
実施日: 2026-06-12

---

## 設計サマリー（前提確認）

| 受取先 | SEN取り分内の% | 補助金採択額に対する% | 500万採択時の実額 |
|-------|-------------|-------------------|----------------|
| 行政書士 | 28% | 7% | 35万円 |
| スケ | 15% | 3.75% | 18.75万円 |
| 仙石 | 57% | 14.25% | 71.25万円 |
| **SEN合計** | 100% | **25%** | **125万円** |

農家の手取り（500万採択時）: 500万 - 125万 = **375万円**

# 🔵 コンテキストAI｜現状の事実を並べる

## 業界標準との比較

| 競合事業者 | 着手金 | 成功報酬 |
|-----------|------|--------|
| さむらい行政書士法人 | 5〜10万円 | 採択額の10〜15% |
| 株式会社セルバ | 10万円 | 採択額の15%（下限150万） |
| 業界平均 | 5〜20万円 | 採択額の10〜20% |
| **SEN設計（今回）** | **ゼロ** | **採択額の25%** |

→ 着手金ゼロ×25%は「業界標準の上限（20%）を5pt超える」。着手金を取らない代わりに成功報酬を高めるモデルとしての設計意図は理解できる。

## キャッシュフローの現実

- 現在のランウェイ: **1.4ヶ月**（50万 ÷ 35万/月）
- 小規模事業者持続化補助金（採択額100万）の場合: SEN取り分 = 25万円
  - 行政書士 7万、スケ 3.75万、仙石 14.25万
  - 申請〜採択通知: 約4〜6ヶ月
- 経営発展支援事業（採択額500万）の場合: SEN取り分 = 125万円
  - 申請〜採択通知: 同様に4〜6ヶ月

**事実**: 7月に契約しても入金は11〜12月。ランウェイ1.4ヶ月の状態でこのタイムラグは、補助金収益をコーチングの代替にできない。

## 法務の事実（2026年1月改正行政書士法）

- 行政書士との正式業務委託契約なしで動けば: 1年以下の拘禁刑または100万円以下の罰金
- 現状: **行政書士は未提携**
- SENの「診断AI・戦略助言・進捗管理」はすべて合法域だが、農家への「書類への口頭赤入れ」すらグレーになるケースがある

## スケの経済合理性（事実）

- スケ現在の報酬: 仙石からの報酬25%（コーチング売上に対して）
- 今回の補助金での取り分: 採択額の3.75%
- 500万採択 → スケ 18.75万円（1案件・工数10〜15時間とすれば時給1.25〜1.87万円）
- これがスケにとって「動くに値するか」は要確認

# 🟡 発散AI｜可能性を広げる

## 25%を「安い」と思わせる価値設計

農家にとって問題は「25%が高いか安いか」ではなく「25%払う価値があるか」だ。SENが他と違う点を言語化する。

### SEN固有の付加価値（他の行政書士・コンサルにはない）

1. **診断AIによる即時マッチング**: 他社が数週間かけて調べる補助金候補を初回面談当日に提示できる。農家の「これって使えるの？」という疑問に即答する体験価値
2. **スケのSNS実績**: リール1本で250万売上。補助金採択後の商品・農場のブランディングまでセットで支援できる。「補助金で機械を買って終わり」ではなく「補助金→設備→EC売上→SNSで集客」の一気通貫が見える
3. **EC×補助金の複合提案**: 小規模持続化補助金はECサイト構築費用が対象。「EC出店費用を補助金で賄い、EC売上を上げる」という農家にとって最も欲しいストーリーを提供できる唯一の存在
4. **着手金ゼロ**: 農家のリスクがゼロ。「採択されなければ1円も払わなくていい」という安心感は、着手金10〜20万を取る競合に対して圧倒的

### 金額感の言い換え戦略

**NG訴求**: 「成功報酬は25%です」
**OK訴求**: 「採択できたら、補助金500万円のうち375万円があなたの手元に残ります。採択できなければ0円です。着手金もいただきません」

→ 「25%を払う」ではなく「375万円を手に入れる」という言語化に変える

### 段階的報酬設計（代替案A）

| 補助金採択額 | 成功報酬% | 農家手取り |
|------------|---------|---------|
| 〜200万円 | 20% | 80%以上 |
| 200〜500万円 | 22% | 78%以上 |
| 500万円〜 | 25% | 75%以上 |

高額補助金ほど25%が正当化しやすい。小額では20%に下げて農家に選ばれやすくする。

### 段階的報酬設計（代替案B）：最初の3件限定15%

- 実績ゼロのうちは15%でいい。最初の3件で「採択率・収益・農家の声」を作る
- 実績ができてから25%に引き上げる
- 「実績を見てから判断してほしい」という姿勢が信頼を生む

# 🔴 否定AI｜穴を刺す

## 穴①：25%は農家に「高い」と言われる

業界標準の上限が20%。SENは実績ゼロで参入する。

- なぜ「実績なし」の25%を農家が選ぶのか、論理が弱い
- 農家コミュニティは横のつながりが強い。「高い業者」という口コミが1件出ると参入した地域全体で信用を失う
- 着手金ゼロを強調しても、農家側は「採択されたら125万も取られる」という計算をする人間は必ずいる

## 穴②：スケへの3.75%は「割に合わない」可能性

スケは現在コーチング売上の25%を受け取っている。
コーチングで79万の案件を決めると、スケの取り分は19.75万円。

補助金サポートで500万採択のケースで、スケの取り分は18.75万円。
ほぼ同じかそれ以下の金額を、補助金の工数（農家との複数回ヤリトリ・進捗管理・SNS連携）で稼ぐことになる。

**スケが「補助金より高単価のコーチングに集中したい」と思っても不思議ではない。**

この報酬設計をスケに提示する前に、スケが納得できるかの個別すり合わせが必要。

## 穴③：行政書士へのオファーとして7%（35万）は魅力的か？

現状の行政書士市場では:
- 着手金10万 + 成功報酬10〜15%が相場
- 自分で営業すれば着手金10万は確定で取れる

SENの提案では:
- 案件を持ち込んでもらう代わりに成功報酬7%のみ
- 着手金なし
- 農家との窓口は仙石・スケが担う

35万円（500万採択時）は魅力的に見えるが、**「案件を持ち込んでもらう価値」が行政書士にとってどれほどか**が鍵。既存顧客が多い行政書士には魅力が薄い。農業補助金に強い行政書士が新規農家を探している状態でないと成立しない。

→ 行政書士選定の条件に「営業力が低く・案件に困っている」という要素を意識的に入れる必要がある。

## 穴④：採択失敗時の逆算をしていない

完全成功報酬モデルの恐ろしさは「採択率×件数」の期待値計算にある。

小規模持続化補助金（採択率64%）で月3件アプローチした場合:
- 採択件数: 1.92件（期待値）
- 不採択件数: 1.08件
- 不採択案件は工数だけかかって収益ゼロ
- 月3件の工数（1件9〜16時間）= 27〜48時間を費やして採択分のみ収益

**これをコーチング営業と並走してスケとともに回せるか、リアルに工数を試算しているか。**

## 穴⑤：「今週の一手」として25%設計を固める前にやることがある

報酬設計を議論する以前に:

1. 行政書士が未提携→法的に動けない
2. スケへの報酬提示・合意なし→チーム設計が空論
3. 農家への無料診断すら1件もやっていない→25%が「高いか安いか」の検証データゼロ

設計を洗練させることに時間をかける前に、「行政書士1人と話す」ことが最優先。設計は実践の中で修正すればいい。

# 🟢 統合AI｜今週の一手

## 「25%設計」の判定

**現時点では「妥当だが、証明できていない」設計。**

25%という数字自体は、SENの付加価値（AI診断 + EC連携 + SNS + 着手金ゼロ）が実際に機能するなら正当化できる。しかし現状は:

- 実績: ゼロ
- 行政書士: 未提携
- 農家への診断: 未実施
- スケへの合意: 未取得

「25%は適切か」という問い自体が早い。今は「25%を証明できる体制を整える」フェーズ。

## 今週の一手（具体的行動・優先順位付き）

### ① 行政書士へのコンタクト（今週水曜までに必達）

**やること**: X（旧Twitter）で「補助金 行政書士 農業」で検索し、農業系補助金を発信している行政書士を3名ピックアップ。DMを送る。

**DM文面（そのまま使える）**:
```
はじめまして。農業経営者向けに補助金申請サポートを展開している
仙石と申します。弊社では農業特化のAI補助金診断と、EC・SNS支援を
組み合わせた集客から採択後支援まで一気通貫のサービスを準備しています。

農家への無料診断から案件化した後、書類作成・申請代理をお願いできる
行政書士との業務提携を検討しています。

成功報酬型（採択額の7%）で案件を持ち込む形式を考えています。
一度お話できますでしょうか。
```

### ② スケへの報酬設計の共有（今週中）

「補助金サポートに一緒に動いてほしい。役割はSNS農家開拓と進捗管理。取り分はSENの15%（採択額の3.75%）。500万採択なら約19万。コーチングと並走だから今すぐ主力にはしない」をそのまま話す。

**確認すること**: スケがこれをやりたいかどうか、1点だけ。合意できなければ仙石1人 + 行政書士の2者体制に設計を変える。

### ③ 25%の数字は「最初の3件は20%に下げる」に修正を検討

理由: 実績ゼロで25%を農家に提示した際、「高い」という反応が出た場合に対処法がない。最初から「実績ができるまで20%」にしておくと:

- 農家への説明が楽になる（業界標準の上限内）
- 最初の採択実績が出てから25%に引き上げる際に「成果を出したから」という根拠ができる
- 農家への心理的障壁が下がり、初案件を取りやすくなる

**修正後の設計（初期3件限定）**:

| 受取先 | 補助金額に対する% | 500万採択時の実額 |
|-------|----------------|----------------|
| 行政書士 | 7% | 35万円（変わらず） |
| スケ | 3% | 15万円 |
| 仙石 | 10% | 50万円 |
| **SEN合計** | **20%** | **100万円** |

農家の手取り（500万採択時）: 400万円 → 農家への訴求が「500万のうち400万が手元に残る」になる

### ④ 今週やらないこと

- 25%か20%かの最終決定（農家の反応を見る前に確定させる必要はない）
- ECサイト・AI診断ツールの開発（行政書士なしでは使えない）
- 農業補助金の詳細学習（動きながら学ぶ。今は行政書士に聞けばいい）

## 統合AIの最終見解

25%という数字の前に解くべき方程式がある。

**行政書士がいない = 法的に動けない = 25%も20%も関係ない。**

今週の優先順位は1つ。**行政書士に1通DMを送ること。** それだけ。

報酬設計の細部を議論し続けることは「プロセスに酔う」パターンと同じ。動いた後に修正する。止まったまま完璧を目指さない。
