- **申請難易度**: ★☆☆（農業大学校・農家研修への参加が条件。書類はシンプル）

### B. 経営開始資金（農林水産省）
- **補助上限**: 月13.75万円（年165万円）× 最長3年。夫婦共同経営は1.5倍
- **補助率**: 定額
- **採択率**: 要件審査型（独立就農後5年以内が条件）
- **申請難易度**: ★☆☆（都道府県窓口経由。書類作成は中程度）

### D. 産地生産基盤パワーアップ事業（農林水産省）

## 2. 補助金申請サポートの相場

### 行政書士・認定支援機関の標準報酬体系

**着手金**:
- 相場: 5万円〜20万円
- 小規模事業者持続化補助金: 5万円前後
- ものづくり補助金・スマート農業補助金: 10万〜20万円
- 認定支援機関経由の場合: 10万〜15万円

**成功報酬**:
- 業界標準: **補助金採択額の10〜20%**
- さむらい行政書士法人: 着手金5〜10万円 + 採択額の10〜15%
- 株式会社セルバ: 初期費用10万円 + 採択額の15%（下限150万円）
- 税理士法人アピロ: 着手金5〜10万円 + 採択額の5〜15%

**完全成功報酬型**:
- 着手金ゼロ（預り金5万円）+ 採択額の25%（不採択時は全額返金）というモデルも存在
- 農業系でこのモデルを打ち出すと差別化になる

**計算例**:
- 農家が経営発展支援事業（上限500万円）を申請 → 採択額500万円
- サポート業者の収入: 着手金10万 + 成功報酬10%=50万 → 合計60万円
- 農家の実質負担: 60万円（補助金500万のうち60万を費用に充てる。差引き手取り440万円）

**農業特化 vs 中小企業一般の費用差**:
- 現状、農業特化サポートを明示する事業者は少数。一般的な中小企業向けコンサル料金と大差ない
- 農林水産省独自の制度が多く、制度知識で差別化できるため将来的には農業特化プレミアムが発生しやすい

**認定支援機関の報酬設計**:
- 認定支援機関（経営革新等支援機関）の年間顧問料: 20万円前後
- 報酬が安い機関ほど採択率が低い傾向がある

## 3. 行政書士法との境界線（法律的整理）

### 2026年1月1日施行の改正行政書士法の核心

**改正行政書士法第19条（旧条文からの変更点）**:
- 従来: 「報酬を得て、官公署に提出する書類を作成することを業とすることができない」
- 改正後: 「他人の依頼を受け**いかなる名目によるかを問わず**報酬を得て」官公署提出書類を作成することは行政書士のみが可能

「いかなる名目によるかを問わず」という文言追加が最大のポイント。コンサルティング料・会費・顧問料・指導料など、どんな名目でも実態が書類作成であれば違法。

**罰則（改正後）**: 1年以下の拘禁刑または100万円以下の罰金 + 両罰規定（法人にも罰金）

### 行政書士でなくてもできること（合法）

1. 補助金制度の概要説明・情報提供（公募要領の読み方解説を含む）
2. 事業計画の方向性へのアドバイス・壁打ち
3. ビジネスモデルの改善提案
4. 申請者が自分で書いた文章の「誤字脱字の指摘」レベルの確認
5. 「この部分をもう少し明確に記載すべき」という口頭・文書での指導

**総務省グレーゾーン解消制度（2022年回答）**:
「相談の範疇である限りにおいて、行政書士の独占業務とされている書類の作成には当たらない」

### 行政書士でないとNGなこと

1. 補助金申請書・事業計画書の作成・代筆（有償）
2. 「完成版」の申請書類を作成して農家に渡すこと
3. 農家のPCに申請書を打ち込む入力代行（有償）
4. どんな名目であれ実質的な書類作成代行

### グレーゾーン（実務上の判断ライン）

| 行為 | 判定 |
|-----|------|
| 農家が書いたドラフトを「方向性が違う、こう直すべき」と口頭で伝える | ✅ 合法 |
| テンプレートを農家に渡して「この空欄を埋めてください」と説明する | △ 合法寄りのグレー |
| 農家が書いたドラフトを受け取って赤入れして返す | △ 実態次第でグレー |
| テンプレートを自分で埋めて農家に渡す | ❌ 違法 |
| 農家のPCを操作して申請書を入力する | ❌ 違法 |

### 結論

**2026年以降の適法モデル = 「戦略コンサル + 行政書士のペア体制」が必須**
- コンサル側: 事業計画策定支援、補助金マッチング診断、申請戦略助言（有償OK）
- 行政書士側: 書類作成・申請代理（有償。これが独占業務）

## 4. 行政書士提携モデルの設計

### 業務分担の具体例

| フェーズ | コンサル（仙石側）の役割 | 行政書士の役割 |
|---------|---------------------|------------|
| 初回面談 | 農家のニーズ・事業内容のヒアリング | 不参加でよい |
| 補助金診断 | 最適な補助金の選定・説明 | 不参加でよい |
| 戦略設計 | 事業計画の方向性・付加価値の言語化支援 | 不参加でよい |
| 書類作成 | 資料提供・口頭アドバイスのみ | **書類を実際に作成（独占業務）** |
| 申請 | 進捗管理・スケジュール管理 | 申請代理 |
| 採択後 | 報告書内容確認・次回補助金提案 | 実績報告書作成 |

### 報酬分配の現実的な設計

**モデルA（一般的）**:
- 農家への請求: 着手金10万円 + 採択額の15%
- 行政書士への支払い: 着手金5万円 + 採択額の5〜7%
- コンサル手残り: 着手金5万円 + 採択額の8〜10%

**モデルB（仙石向けの農業特化型推奨）**:
- 農家への請求: 着手金ゼロ + 採択額の20%（完全成功報酬。農家にリスクゼロ訴求）
- 行政書士への支払い: 採択額の7%
- コンサル手残り: 採択額の13%

> ただしランウェイ1.4ヶ月の現状では完全成功報酬は危険。初期は着手金3〜5万を取ること。

### 紹介料・業務委託契約の注意点

- 行政書士倫理第12条: 行政書士は依頼のあっせんに対する報酬の授受を禁止
- **実務的対処**: 「紹介料」という名目は避け、「業務委託料」として契約書に明記する
- 行政書士への支払いは「書類作成・申請代理業務の対価」として支払う形を取る
- 農家との契約はコンサル（仙石側の法人）が行い、行政書士は下請け業務委託の形にする

### 提携行政書士の探し方・条件

**探し方**:
1. 日本行政書士会連合会の名簿検索（https://www.gyosei.or.jp/）
2. 農業・補助金実績を持つ行政書士をX/Twitterで検索
3. 農業系の士業交流会・SNSコミュニティでの接触

**条件の優先順位**:
1. 補助金申請実績（農業系であれば理想的）
2. 成功報酬型でも動ける柔軟さ
3. 返信速度・コミュニケーション能力
4. 認定支援機関資格保有（ものづくり補助金等に対応可能になる）

## 6. 診断AI・自動化ツールを使った場合の優位性と法務注意点

### AIによる補助金マッチング・診断の市場事例

- **株式会社Salud「補助金の窓口 AIマッチング診断ツール」**（2026年4月開始）: 企業情報を入力するだけで最適補助金を特定し申請戦略まで自動生成
- **株式会社Mycat「補助金1分マッチング診断」**: 業種・従業員数・活用目的の3問に答えるだけでマッチング。無料
- **情報の泉（j-izumi.com）**: 20,000件以上の補助金情報をAIでリアルタイムマッチング。住所・業種・資本金・従業員数から自動診断

### AI活用による差別化

- **速度**: 農家の基本情報（作物・規模・地域・設備状況）を入力するだけで5〜10分でマッチング → 従来の専門家相談より圧倒的に速い
- **精度**: 農林水産省・経済産業省の補助金を横断的に検索 → 見落とし防止
- **コスト**: AI診断自体は無料〜低価格で提供可能 → 集客ツールとして使える
- **差別化戦略**: 「農業特化AI補助金診断（無料）」を入口に、実際のサポート（有償）へ誘導するリードジェネレーション

### AI出力を「申請書類」に使う際の法的注意点

| パターン | 判定 |
|---------|------|
| AIが生成した文章を「参考情報」として渡し、農家が自分で打ち直す | △ グレー（実態次第） |
| AIが生成した文章を行政書士が確認・責任を持って書類化する | ✅ 合法 |
| AIが生成した文章を農家に渡して「そのまま申請に使ってください」 | ❌ 違法寄りのグレー |

**結論**: AIは「診断・マッチング・情報提供」ツールとして使い、「書類作成」は行政書士に委託する設計にする。AIが作成 → 誰が有償でその書類を提供したかが問われる。

## 7. 農家1件あたり収益シミュレーション

### パターンA: 保守シナリオ

- **対象補助金**: 小規模事業者持続化補助金（採択額100万円）
- **採択率**: 64%（実データ）
- **農家への請求**: 着手金ゼロ + 採択額の20% = 20万円
- **行政書士への分配**: 採択額の7% = 7万円
- **仙石の手残り**: 13万円（採択した場合）
- **期待値**: 13万円 × 64% = **8.3万円/件**
- **工数**: ヒアリング2h + 戦略設計3h + 行政書士調整2h + 採択後報告2h = **約9時間**
- **時給換算**: 約9,200円/時間

### パターンB: 標準シナリオ

- **対象補助金**: 経営発展支援事業（採択額500万円）
- **採択率**: 50%（仮定）
- **農家への請求**: 着手金10万円 + 採択額の15% = 85万円
- **行政書士への分配**: 着手金5万 + 採択額の5% = 30万円
- **仙石の手残り（採択時）**: 55万円
- **期待値**: 着手金5万（確定） + 55万×50% = **32.5万円/件**
- **工数**: 約16時間
- **時給換算**: 約20,000円/時間

### パターンC: 楽観シナリオ

- **対象補助金**: スマート農業導入支援（採択額1,500万円）またはものづくり補助金（採択額1,000万円）
- **採択率**: 40%（ものづくり補助金実績ベース）
- **農家への請求**: 着手金15万円 + 採択額の12% = 135〜195万円
- **行政書士への分配**: 着手金5万 + 採択額の4% = 45〜65万円
- **仙石の手残り（採択時）**: 90〜130万円
- **期待値**: 着手金10万（確定） + 110万×40% = **54万円/件**
- **工数**: 約24時間
- **時給換算**: 約22,500円/時間

## 8. 致命的リスク7つと対策

### リスク1: 行政書士法違反（最大の法務リスク）
**内容**: 2026年1月施行の改正行政書士法により、書類作成代行は無資格者に完全禁止。罰則は1年以下の拘禁刑または100万円以下の罰金。法人への両罰規定もある。
**対策**: 行政書士との正式な業務委託契約を締結し、書類作成は行政書士が担当する体制を必ず整える。一切の書類への「書き込み・代筆」を自分でしない。

### リスク2: 不採択時の農家クレーム・返金要求
**内容**: 農家が着手金を払った後に不採択になった場合、「詐欺」と言われるリスク。農家は口コミが命のコミュニティ社会のため、1件のトラブルが全体に波及する。
**対策**: 契約書に「採択保証はできない」旨を明記。完全成功報酬型にすることで農家のリスクをゼロにする設計。書面での契約必須。

### リスク3: 補助金制度の改廃
**内容**: 農林水産省の補助金は毎年度予算・制度が変わる。経営継承・発展等支援事業は令和7年度で終了予定。
**対策**: 特定の補助金に依存しない。月1回の農林水産省・中小企業庁の公募情報チェックをルーティン化する。

### リスク4: 収益の不安定性（採択前後の時間差）
**内容**: 補助金申請から採択通知まで2〜6ヶ月かかる場合が多い。成功報酬型の場合、その間に収入がゼロになる。ランウェイ1.4ヶ月の現状では致命的。
**対策**: コーチング事業との収益の二本柱を維持する。補助金サポートは少額でも着手金を取る設計にする（少なくとも最初の数件は）。

### リスク5: 農家のITリテラシー不足
**内容**: 農家は電子申請（jGrants等）に不慣れな場合が多い。「書類を書く」以外のサポート工数が膨らむ。
**対策**: オンボーディングの工数を事前に見積もる。複雑な電子申請を含む補助金は報酬を高く設定するか、行政書士に一括委任する。

### リスク6: 競合の価格競争
**内容**: 参入障壁が低いため、行政書士・税理士・中小企業診断士が農業補助金サポートに参入しやすい。2026年法改正で行政書士は独占業務が強化され、無資格コンサルに対して優位になる。
**対策**: 「書類作成」で戦わない。EC・SNS・コーチングを組み合わせた「農家の売上を上げる」という上位の価値提供で差別化する。

### リスク7: 農家との信頼関係崩壊
**内容**: 農業コミュニティは横のつながりが強く、1件の失敗・トラブルが口コミで広がりやすい。
**対策**: 最初の1〜3件は採択率60%以上の小規模事業者持続化補助金から着手する。実績を積んでから高額補助金へ移行する。

### Week 1: 知識・提携の基盤構築

- 農林水産省の補助金公募ページをブックマーク、全補助金一覧をスキャン（3時間）

- 行政書士1名と初回面談・条件すり合わせ

- なのか・さとなに「農家の知り合いで設備投資・機械購入を考えている人いない？」とヒアリング

### Week 3: 初案件の獲得

- なのか・さとなのパイプで農家1〜2件にアプローチ
- 無料の補助金診断を実施（1〜2時間。農家の設備状況・売上・規模・課題を聞く）
- 最適な補助金を提案（公募要領と照合）
- 農家が興味を持てばサービス契約締結（着手金3〜5万円 + 採択額の15%を提案）

- 行政書士に案件情報を引き渡し

- 2件目以降の農家へのアプローチ継続
- 農林水産省の次期公募スケジュールを確認し、申請タイミングから逆算したスケジュール管理

## 10. 抜け道・組み合わせ戦略

### A. 人材紹介（農業人材マッチング）との組み合わせ

**モデル**:
- 補助金サポートで農家と信頼関係を構築
- 「設備を補助金で揃えた → 次は人手が足りない」という次のニーズを人材紹介で受ける
- さとな（北海道在住）を通じた北海道農家ネットワーク × 人材マッチング
- 収益: 人材紹介手数料（採用者年収の20〜35%が相場）

**注意**: 有料職業紹介事業には厚生労働大臣の許可が必要（職業安定法）。無許可の有料職業紹介は違法。

### B. 農業EC（産直販売）との組み合わせ

**一気通貫モデル**:
1. 補助金サポート → 農家が設備投資・ブランド強化に成功
2. 採択後の農家に「次はこの設備・商品をECで売りましょう」と提案
3. 仙石のEC構築スキル・なのかのSNS発信力で農家のオンライン販売を支援
4. EC売上の数%をロイヤルティとして受け取る（継続収益化）

**補助金との相性が良い組み合わせ**:
- 小規模事業者持続化補助金: EC出店費用・パッケージ改良が対象
- みどりの食料システム法の認定を受けた農家は各種補助金で優遇措置

### C. AIサポートツールとしての展開

**「農業補助金AI診断ツール」の開発・販売**:
- 農家が自分の情報を入力するだけで最適な補助金を5分で診断
- 無料版（診断のみ）→ 有料版（申請支援サービスへの接続）
- Claude API等を活用したLINE Bot型の診断ツールも可能
- 将来的には行政書士との連携システムとして他のコンサルへのBtoB展開も可

**法的注意点**: ツール自体の開発・販売は適法。ただしツールが「申請書類を生成する」機能を持ち、有償で提供した場合は行政書士法違反になりうる。「情報提供・マッチング」に機能を限定する必要がある。

### D. コーチング事業（本業）との掛け合わせ

**農家コーチング + 補助金サポートの複合商品**:
- 現状のコーチング（CORE79万/MASTER132万/LEGACY275万）に農業経営者向けプランを追加
- 「農業経営者向けMASTERプラン（132万円）= 半年間の経営コーチング + 補助金申請サポート（成功報酬別）」
- 経営者としてのマインドセット × 補助金という実利の組み合わせで高単価化

### E. 補助金 × EC × SNS × コーチングの複合モデル設計

```
【フロント】無料農業補助金診断（SNS・なのかのリール経由で集客）
     ↓
【コア】農業経営コーチング（132〜275万円）
     ↓ 内包されるサービス
  ├─ 補助金申請サポート（行政書士提携・成功報酬）
  ├─ EC構築支援（農産物オンライン販売のセットアップ）
  └─ SNS発信代行・指導（なのかとの協業）
     ↓
【継続】EC売上ロイヤルティ + 次年度補助金サポート（リピート）
```

**この複合モデルの収益試算（農家1社あたり・初年度）**:

| 収益源 | 金額 |
|-------|------|
| コーチング | 132万円（一括） |
| 補助金成功報酬 | 13〜65万円（補助金規模による） |
| EC構築支援 | 20〜50万円 |
| 継続EC管理 | 3〜5万円/月 |
| **1社あたり初年度収益** | **168〜262万円 + 継続3〜5万円/月** |

**借金290万との関係**:
- この複合モデルで2〜3社成約すれば借金完済が視野に入る
- ただし採択から入金まで2〜6ヶ月のタイムラグあり。コーチング収益を先行させること

## 総括：仙石漱が参入すべきかの現実的評価

### 参入すべき理由

1. **農家パイプが既にある（なのか・さとな）**: 0から農家を開拓する必要がない
2. **法人格あり**: 業務委託契約・サービス販売が即日可能
3. **コーチング×補助金の掛け合わせ**は業界にほぼ存在せず、差別化余地が大きい
4. **初期投資5〜13万円**: 低コストで参入できる

### 警戒すべき点

1. **ランウェイ1.4ヶ月が最大のリスク**: 補助金サポートは入金まで2〜6ヶ月かかる。今すぐやるなら着手金型にして先に現金を確保しないと資金ショートする
2. **2026年1月施行の行政書士法改正を甘く見るな**: 書類1枚でも代わりに書けば1年以下の拘禁刑。行政書士提携なしで1人で動き始めることは禁止
3. **コーチング営業の自動化が最優先**: 月売上300万安定が先。補助金サポートはその「次の柱」として、月1〜2件から静かに始める
