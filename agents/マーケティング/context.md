**高卒・24歳以下に完全絞り込み**

| 項目 | 内容 |
|------|------|
| 年齢 | 〜24歳 |
| 学歴 | 高卒・中退・大学不問だが動いている人 |
| 状態 | 起業・副業・フリーランスを目指して動いているが詰まっている |
