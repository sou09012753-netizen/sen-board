## チームメンバーの現状

### なのか（インターン・京都）※ 2026-07-01 更新

| 項目 | 内容 |
|------|------|
| 役割 | SNS・新規開拓担当（農家開拓・撮影・Instagram/TikTok運用） |
| **現在の状態** | **2026-07-01 MTG実施済み・復帰確認済み** |
| 動機づけ | 無能バレ恐怖型・フィールドタイプ。リモートで消える傾向あり。週1で必ずこちらから確認。第三者（農家）からの「ありがとう」が最大の報酬 |
| 動く条件 | 現場に行ける・「面白い」という巻き込み型の声かけ |
| 止まる条件 | できないことが可視化される・管理・リモートでの細かい確認 |
| 報酬 | 完全成果報酬 ＋ 初期撮影ボーナス（3軒×5,000円） |
| 経費 | 公共交通機関の実費をSENが負担（事前LINE承認制） |
| マネジメント方針 | 週1で必ずこちらから確認。三者面談のセッティングが再開のトリガー |

### さとな（インターン・北海道）※ 2026-07-01 更新

| 項目 | 内容 |
|------|------|
| 役割 | 農業EC運営担当（商品登録・注文管理・SOP作成・バックオフィス） |
| 動機づけ | 承認欲求型。未来との接続を先に見せる。成果を月次で数字として言語化して伝える。過度な期待は禁止 |

| マネジメント方針 | 「これが将来こうつながる」を見せる。承認欲求が高いので公開して褒める場を作る |
