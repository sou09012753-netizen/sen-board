# 人材AI — context

人材AIがマネジメント判断をするための情報。「誰が・どう動くか・何がリスクか」の前提。

## チームメンバーの現状

### スケ（右腕）

| 項目 | 内容 |
|------|------|
| 役割 | 営業補佐・全体サポート |
| 報酬 | 成約時20%（= 約15.8万/件） |
| リスク | 報酬20%に不満あり |
| 未確認 | 「長期で一緒にいたい、そのために何が必要か」を直接聞いていない |
| 対策 | 囲い込みは短期では機能するが長期では亀裂になる。早急に本音を聞く |

### なのか（インターン・京都）

| 項目 | 内容 |
|------|------|
| 役割 | 農家SNSインフルエンサー・看板娘 |
| 動く条件 | 現場に行ける・「面白い」という巻き込み型の声かけ |
| 止まる条件 | できないことが可視化される・管理・リモートでの細かい確認 |
| マネジメント方針 | 「できているか」確認は萎縮させる → 「これ面白くない？」で動かす |

### さとな（インターン・北海道）

| 項目 | 内容 |
|------|------|
| 役割 | 農業EC編集・企画 |
| 動く条件 | 将来とのつながりが見える・小さい成果を褒められる場がある |
| 止まる条件 | 期待値を上げすぎる・ただの作業指示 |
| マネジメント方針 | 「これが将来こうつながる」を見せる。承認欲求が高いので公開して褒める場を作る |

## リスクレベル

| メンバー | リスク内容 | 優先度 |
|---------|-----------|--------|
| スケ | 報酬不満・長期意思未確認 | 高 |
| なのか | リモート管理で消える可能性 | 中 |
| さとな | 期待ギャップで逃げる可能性 | 中 |

## 仙石漱のマネジメント上の癖（人材AIが監視する）

- 孤独が怖くてコントロールしようとする癖が出ると、スケ・なのか両方に悪影響
- 本音を話せる相手がいない状態が続くと危険
