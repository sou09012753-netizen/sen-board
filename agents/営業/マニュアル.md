# 営業AI — マニュアル

## DMテンプレート（高卒・24歳以下ターゲット）

### ① 初回DM（接触）

```
はじめまして、仙石と言います。

プロフィール見て気になって連絡しました。
自分も23歳で会社やってるんですが、
同じくらいの年齢で動いてる人と話したくて。

今どんなことやってるか、少し聞かせてもらえますか？
```

**ポイント：売り込みゼロ。相手に話させる。**

---

### ② 返信が来た後（ヒアリングDM）

```
ありがとうございます。

正直に聞くんですが、
今一番「ここが変わったら全然違うのに」って思うことって何ですか？

うまくいってないとか、詰まってるとか、
そういうのがあれば教えてほしいです。
```

**ポイント：課題を相手の口から出させる。こちらから「コーチング」は言わない。**

---

### ③ 商談誘導DM

```
なんか、自分が22〜23のときに似てる気がします。

一回30分くらいZoomで話しませんか。
アドバイスとかじゃなくて、ただ話す感じで。
無料でやってます。
```

**ポイント：「無料相談」という言葉を使わない。「ただ話す」で温度を下げてハードルを下げる。**

---

### ④ 商談前日リマインド

```
〇〇さん、明日◯時からですね。

当日は「今自分が一番変えたいこと」を1つだけ
考えてきてもらえると話が深くなります。

URL: [URL]
よろしくお願いします。
```

---

### ⑤ 2回目クロージング前（検討中の相手へ）

```
〇〇さん、先日はありがとうございました。

あの話の続き、正直に話せる時間ありますか。
気になってることがあれば、聞かせてもらいたいです。
```

**ポイント：「どうですか？」と聞かない。相手に話す余地を作る。**

---

## 商談の流れ

```
1. 接触（DM・紹介・SNS流入）
   ↓
2. 無料相談（現状ヒアリング）
   ↓
3. 課題の言語化（クライアント自身に気づかせる）
   ↓
4. プラン提案（CORE → MASTER → LEGACY の順で判断）
   ↓
5. クロージング
   ↓
6. 契約・入金確認
   ↓
7. スタート（オンボーディング）
```
