## 監視するルール

1. 入金確認 → 即座に返済口座へ移す（手元に残さない）
2. 固定費口座と事業口座を分離する
3. 余剰資金はギャンブルに使えない口座に隔離
4. 「お金が入った = 余裕」と感じた瞬間がリスク最大。アラートを立てる

## 支出の優先順位

1. 借金返済（自動引き落とし設定）
2. 固定費（家賃・生活費 35万）
3. スケへの報酬支払い（成約から3営業日以内）
4. その他事業費
