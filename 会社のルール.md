# 会社のルール — 事業運営規定

最終更新: 2026-06-11（報告ルール追加）
対象: 仙石漱・スケ（右腕）・インターン生全員

> このファイルは「決めたこと」の置き場。守れなかった場合の記録も残す。

## 1. 財務ルール

### 1-1. 入金処理フロー（絶対に守る）

```
成約・入金確認
  ↓
① 翌営業日中に返済口座へ移す（手元に置かない）
  ↓
② スケへの報酬を支払う（入金確認から3営業日以内）
  ↓
③ 固定費口座に当月分を確保（35万）
  ↓
④ 残額は事業費口座へ
```

### 1-2. 口座分離ルール

| 口座 | 用途 | ルール |
|------|------|--------|
| 返済専用口座 | 借金290万の返済のみ | 入金のたびに最優先で移す |
| 固定費口座 | 家賃・生活費（月35万） | 毎月1日に自動振替設定 |
| 事業費口座 | 広告・外注・ツール費用 | 使途を記録する |
| 手元口座 | 日常の支出 | 上限10万まで。超えたら事業費から補填しない |

### 1-3. 現在の財務制約

- 借金残高: **290万**
- 完済の鍵: **月2件成約**（月2件 = 月収63万×2 = 126万 → 固定費35万引いて91万返済可能）
- ランウェイ: **約1.4ヶ月**。今月1件決まらなければ資金繰りが厳しくなる

### 1-4. ギャンブル禁止ルール

- 余剰資金が出た瞬間に即座に別口座へ移す
- 「今月余裕がある」と感じたらスケに即連絡する
- ギャンブルができる物理的状態を作らない（キャッシュを手元に置かない）

## 2. 営業ルール

### 2-1. 商品・単価

| プラン | 単価 | 状態 |
|--------|------|------|
| 主力プラン | 79万 | **唯一の使用プラン** |

> 132万・275万は未確定のため、確定するまで一切使用しない。

### 2-2. クロージングルール

- クロージングは**最大2回**
- 2回目で決まらない場合は「手放す」。3回目は行わない
- 1回目クロージング後の返事待ちは最大5日。5日経過で追撃1回のみ

### 2-3. 報酬分配

| 項目 | 金額 |
|------|------|
| 単価 | 79万 |
| スケ取り分（20%） | 約15.8万 |
| 会社取り分（80%） | 約63.2万 |
| 支払いタイミング | 入金確認から**3営業日以内** |

### 2-4. 商談のルール

- 無料相談は現状ヒアリングが目的。売ろうとしない
- 「客が本当に求めるものを満たせる」と判断した時だけ提案する
- 嘘・誇大表現・実績の盛り込みは禁止
- ノーショー後の再アポは1回まで。2回ノーショーなら対象外

### 2-5. 今月の最低ライン

- **今月クロージング1件を確実に決める**（現在2回目クロージング1人・初回3人のパイプライン）
- 月2件ペースを来月以降の標準にする

## 3. チーム・外注ルール

### 3-1. スケ（右腕）

| 項目 | ルール |
|------|--------|
| 報酬 | 成約時20%（= 約15.8万）。支払いは入金から3営業日以内 |
| 報酬交渉 | 「長期で一緒にいたい。そのために何が必要か」を直接聞く（未実施。最優先） |
| 指示の仕方 | 1日1回の定例連絡。詰め込みすぎない |
| 契約 | 業務委託契約書を正式に締結する（現在口頭。法務ルール参照） |

### 3-2. なのか（インターン）

| 項目 | ルール |
|------|--------|
| 指示の仕方 | 「できているか」確認は禁止 → 「これ面白くない？」の巻き込み型 |
| 仕事のアサイン | 現場に行ける仕事を優先。リモートの細かい管理タスクは渡さない |
| 失敗時 | 指摘しない。別の切り口で再挑戦の場を作る |

### 3-3. さとな（インターン）

| 項目 | ルール |
|------|--------|
| 指示の仕方 | 「これは将来こうつながる」をセットで伝える |
| 期待値 | 最初から高い期待値を伝えない。小さい成功体験から積み上げる |
| 承認 | 成果が出たら公開して褒める場を作る（Slack投稿・定例での紹介等） |

### 3-4. 共通ルール（全員）

- タスクは「誰が・いつまでに・何を」の3点がなければ依頼しない
- 締め切りのない依頼は存在しない扱い
- 1人で抱え込んでいるサインが出たら仙石漱が先に声をかける

## 5. マーケティングルール

### 5-1. 現在フェーズの方針

- コーチング集客が最優先。農業ECマーケは借金完済後まで本格始動しない
- DMは量より質。ノーショー率36%を改善するナーチャリングを入れる

### 5-2. SNS投稿ルール

| チャネル | 頻度 | 担当 |
|---------|------|------|
| X | 未定義 → 今月中に設定 | 仙石漱 or スケ |
| Instagram | 未定義 | なのか（農家アカウント） |
| note | 未定義 → 今月中に設定 | 仙石漱 |

### 5-3. ノーショー対策ルール

- 商談予約後、**前日と当日朝にリマインドを送る**
- リマインドを送っても返事がない場合はノーショーとして処理
- ノーショーは1回まで再アポを受ける。2回は対象外

## 6. 法務・契約ルール

### 6-1. 契約書必須ルール（今すぐ整備）

- コーチングを成約した場合は**必ず契約書を交わしてから入金を受ける**
- 口頭合意のみでの入金受け取りは禁止
- スケとの業務委託契約書を**今月中に締結する**

### 6-2. コーチング契約書に含めるべき最低限の項目

- [ ] 契約期間・開始日
- [ ] 料金・支払い方法・支払い期日
- [ ] キャンセルポリシー（返金の可否・条件）
- [ ] 秘密保持（NDA）
- [ ] 成果の保証に関する免責事項

### 6-3. キャンセルポリシー（案・要確定）

- 開始前7日以上: 全額返金
- 開始前7日未満: 50%返金
- 開始後: 返金なし

> ※ 上記は案。実際の運用前に仙石漱が確定してこのファイルに記載する。

## 7. 意思決定ルール

### 7-1. 仙石漱が1人で決めてはいけないもの

- 50万以上の支出（スケに相談してから）
- 新しいプランの価格設定
- チームメンバーの雇用・解除

### 7-2. 判断の前に必ず問うこと

1. これは3年後の10億（農業EC）に直結するか？
2. 借金完済の前後どちらのフェーズのアクションか？
3. 熱量が下がった状態で判断していないか？
4. 嘘や誇大が含まれていないか？

### 7-3. 迷ったときの優先順位

```
借金返済 > 既存クライアントへの提供価値 > 新規獲得 > 農業EC > それ以外
```

## 8. 報告ルール

### 8-1. 成果物の保存

- **成果物は必ず `成果物/` フォルダに保存する**
- 仕上がったものをチャットや口頭だけで共有して終わりにしない

### 8-2. ファイル名のルール

```
形式: YYYY-MM-DD_誰が読んでも内容がわかるタイトル.拡張子

例:
  2026-06-11_コーチング契約書テンプレート.pdf
  2026-06-11_スケ業務委託契約書.docx
  2026-06-15_なのか農家SNS投稿スケジュール6月.xlsx
  2026-06-20_SENアプリSESSION0設計書.md
```

### 8-3. ファイル名でやってはいけないこと

| NG例 | 理由 |
|------|------|
| `契約書.pdf` | いつのものか不明 |
| `最終版.pdf` | 最終版が複数生まれる |
| `スケ用.docx` | 何の書類か不明 |
| `修正2.xlsx` | 何を修正したか不明 |

### 8-4. 成果物の種類と置き場所

| 種類 | 置き場所 |
|------|---------|
| 契約書・請求書 | `成果物/契約・法務/` |
| 営業資料・提案書 | `成果物/営業/` |
| SNS投稿・コンテンツ | `成果物/マーケティング/` |
| アプリ・ツールの設計書 | `成果物/プロダクト/` |
| その他（分類不明） | `成果物/` 直下に置いて後で整理 |
